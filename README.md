# NFL Player Statistics Analysis

This program reads through csv files, cleans the data, and analyzes player statistics. 
It ranks players by different metrics, and can generate full rankings, or specific rankings for individual players.


## Example Output

The program can output top player rankings for many different stats:

(Rushing Yards)
```
James Cook III (BUF)     1,621
Derrick Henry (BAL)     1,595
Jonathan Taylor (IND)     1,585
Bijan Robinson (ATL)     1,478
De'Von Achane (MIA)     1,350
Kyren Williams (LAR)     1,252
Jahmyr Gibbs (DET)     1,223
```

It can also output specific player stats and their rankings:

(Jahmyr Gibbs)
```
Name: jahmyr gibbs
Rank: 3

Rush Attempts: 243  (11)
Rush Yards: 1,223  (7)
Rush Y/A: 5.0  (14)
Rushing Touchdowns: 13  (3)
```


## Skills Demonstrated

- Python programming
- Reading and parsing CSV files
- Data cleaning and numeric conversion
- Sorting and ranking algorithms
- Generating player stat summaries
- Basic data analysis logic
